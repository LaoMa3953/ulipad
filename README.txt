运行环境
=====================================================================================
python 2.3.3
wxPython 2.4.2.4+ Unicode版本




安装
=====================================================================================

直接将下载的包解压到一个目录下即可。如果是覆盖安装，请删除原来所有的pyc文件。




运行
=====================================================================================

执行 python newedit.pyw 或 python newedit.py 均可。





生成帮助文档
=====================================================================================

帮助文档存放于doc目录下，全部使用reStructuredText格式编写。
.txt为原始文档，.html为生成后的文档。如果你有兴趣自已生成html文档，要安装docutil包，使用
它带的html.py生成即可。文档为UTF-8编码。

本软件已经带有全部生成好的html文档，不需要单独生成。
